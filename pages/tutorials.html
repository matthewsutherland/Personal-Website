Forecasting Toyota Sales in Norway: 2007-2017

Forecasting product demand is an essential aspect of analytics. Ordering too much or too little of a product can minimize a company's profit, leaving too much product unsold or too many customers wanting something that is unavailable. In this post I demonstrate how such an analysis can be undertaken in R using a very simple but effective method for estimating continuous outcome variables. Continue reading.

Postgres SQL in R

This post explains how to query a Postgres SQL database in R on an OSX system. In most instances where SQL queries are being performed, data—usually a set of relations, are being selected from an external database hosting the data. But with no access to a server a Postgres SQL database can be created locally on your own hard drive, which is the approach used in this example. However, if you have access to an external server you can of course use that instead. Otherwise follow this tutorial, which explains how to download and create a Postgres SQL database on your own OSX machine.
